Unduh PostgreSQL melalui tautan ini: https://www.enterprisedb.com/downloads/postgres-postgresql-downloads

Untuk Sistem Operasi Windows, Anda dapat mengunduhnya pada tanda merah di gambar berikut.

Perhatian: Untuk sistem operasi macOS dan Linux dapat mengikuti tutorial instalasi berikut.

  • Install PostgreSQL on macOS
  • Install PostgreSQL on Linux

Langkah 2: Install PostgreSQL

Ikuti langkah-langkah proses instalasi pada PostgreSQL seperti biasa Anda lakukan untuk proses instalasi program lainnya.

Langkah 3: Pilih komponen

Pada bagian komponen ini, ada beberapa hal yang perlu Anda ketahui.

  • PostgreSQL Server: merupakan database server untuk PostgreSQL
  • pgAdmin 4: merupakan tool manajemen database
  • Command Line Tools: merupakan tool yang akan running di Terminal atau CMD, seperti psql, pg_restore, dan sebagainya.
  • Stack Builder: menyediakan GUI dan driver yang bekerja dengan PostgreSQL (perlu koneksi untuk unduh).

Anda tidak memerlukan Stack Builder ini, silakan hapus checklistnya (uncheck), kemudian Next.

Soal 1

Carilah di internet apa saja manfaat Stack Builder dan bagaimana cara menggunakannya?

Langkah 4: Set Password

Kita perlu set password database superuser (default username: postgres). Simpanlah password Anda dengan baik, nanti akan digunakan kembali.

Langkah 5: Set Port Number

Secara default port yang digunakan PostgreSQL adalah 5432. Anda harus memastikan bahwa port tersebut tidak sedang digunakan oleh aplikasi/program lain di Laptop/Komputer Anda.

Klik Next hingga proses instalasi selesai sampai Finish.

Langkah 6: Verifikasi Hasil Instalasi

Ada beberapa cara untuk melakukan verifikasi hasil instalasi PostgreSQL. Anda dapat mencoba mengoneksikan server database PostgreSQL dari aplikasi client seperti psql dan pgAdmin.

Cara tercepat untuk verifikasi instalasi adalah menggunakan program psql.

Klik program psql seperti gambar berikut, maka program command-line (CMD) akan tampil.

Kemudian, ketik informasi yang dibutuhkan seperti server, database, port, username, dan password seperti pada gambar berikut. Untuk tetap menggunakan nilai default yang berada dalam kurung siku, Anda dapat langsung menekan Enter. Kemudian bagian password, Anda ketik sesuai dengan inputan ketika proses instalasi sebelumnya. Jika berhasil tersambung, maka akan tampil seperti postgres=#

Lalu ketik perintah atau query SELECT version(); maka Anda akan melihat hasilnya berupa versi dari PostgreSQL yang telah berhasil Anda install.

Langkah 4: Impor DB

Ketik perintah exit untuk keluar dari psql

Kemudian, navigasikan ke folder bin dari folder instalasi PostgreSQL seperti berikut

Setelah itu, gunakan tool pg_restore untuk memuat data ke database dvdrental:

Dalam perintah ini:
